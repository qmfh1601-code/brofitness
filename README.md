# 브로피트니스 BRO FITNESS 홈페이지

청주 구독형 헬스장 **BRO FITNESS**(용암·금천·복대)의 공식 홈페이지입니다.
밝고 고급스러운 톤 + 주황색 강조 포인트 디자인. 빌드 도구 없이 바로 열리는 React + Tailwind 구조입니다.

---

## 1. 사이트 열어보기 (내 컴퓨터에서)

터미널에서 아래 한 줄을 실행하세요. (맥 기본 내장 파이썬 사용, 설치 필요 없음)

```bash
cd ~/bro-sis-web && python3 -m http.server 8799
```

그다음 브라우저에서 **http://localhost:8799** 접속.
끄려면 터미널에서 `Ctrl + C`.

> 더 간단히: `~/bro-sis-web/실행.command` 파일을 더블클릭해도 서버가 켜집니다.

---

## 2. 내용 바꾸기 (글자·가격·전화·링크)

거의 모든 문구/숫자/링크는 **한 파일**에 모여 있습니다 →
`src/content.js`

- 따옴표 `" "` 안의 **글자만** 바꾸면 됩니다. 쉼표 `,` 와 따옴표는 지우지 마세요.
- 바꾼 뒤 저장하고 브라우저 새로고침하면 바로 반영됩니다.

자주 바꾸는 곳:
| 무엇 | 위치 (content.js 안) |
|---|---|
| 메인 슬로건 | `hero.title` |
| 가격(34,900) | `pricing.main.price` |
| 지점 주소·전화·운영시간·네이버링크 | `branches` |
| 히어로 핵심 지표(34,900 등) | `hero.stats` |
| 트레이너 이름·소개 | `trainers.list` |
| 인스타·앱스토어·카카오 링크 | `brand` |
| 예약 신청 받을 이메일 | `booking.submitTo` |

---

## 3. 사진 / 로고 바꾸기

사진은 모두 `img/` 폴더에 있습니다. **같은 파일명으로 덮어쓰면** 자동 교체됩니다.
예) 메인 대표사진을 바꾸려면 `img/people2.jpg` 를 새 사진(같은 이름)으로 교체.

- 로고를 넣으려면 `img/logo-bro.png` 로 저장 후 `content.js` 의 `brand.logo` 에 경로를
  적으면 텍스트 로고("BRO FITNESS") 대신 이미지 로고가 표시됩니다.
- 현재 사진들은 실제 브로 지점 촬영본으로 채워 둔 임시본이며, 지점별 사진은
  `content.js` 의 `branches[].images` 에서 어떤 지점에 어떤 사진을 쓸지 지정합니다.

---

## 4. 예약·상담 신청은 어떻게 들어오나요?

방문자가 예약 폼을 채우고 **신청하기**를 누르면, 작성 내용이 담긴 메일이
`content.js` 의 `booking.submitTo`(현재 qmfh1601@gmail.com) 로 가도록 메일 앱이 열립니다.
→ 나중에 카카오 채널 상담이나 구글폼으로 바꾸고 싶으면 알려주세요.

---

## 5. 인터넷에 올리기 (실제 배포)

지금은 내 컴퓨터에서만 보입니다. 누구나 접속하는 주소로 만들려면:
- 가장 쉬움: **Netlify Drop** (netlify.com/drop) 에 `bro-sis-web` 폴더를 통째로 끌어다 놓기 → 즉시 주소 생성.
- 또는 Vercel / GitHub Pages.
- 배포 시 현재 개발용 CDN(Tailwind·Babel) 경고는 정식 빌드로 바꾸면 사라집니다. 필요할 때 도와드릴게요.

## 폴더 구조
```
bro-sis-web/
├── index.html          ← 폰트·색상 테마·CDN 설정
├── src/
│   ├── content.js      ← ★ 내용(글자·가격·링크) — 여기만 고치면 됨
│   └── app.jsx         ← 화면 구성/동작 (개발자 영역)
├── img/                ← 사진·로고
└── 실행.command         ← 더블클릭하면 서버 실행
```
